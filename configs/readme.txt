If you want to reset the settings, You're safe to remove xml files in this folder. 
Hearthstone will emit default copies if they get deleted.

gameLangs.xml provides HearthTracker a list of game languages that it can work with.
gameResolutions.xml proide HearthTracker a list of game resolution that it can work with.
heroes.xml provides HearthTracker a list of Heroes that it can scan (must follow the exact order for now)
settings.xml save and store your HearthTracker preferences.
